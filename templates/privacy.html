2. Finalidad del Tratamiento

Sus datos personales serán tratados para las siguientes finalidades:

  • Gestión de pedidos: Para procesar y gestionar los pedidos realizados a través de la tienda online.
  • Atención al cliente: Para responder a consultas, solicitudes y proporcionar soporte técnico.
  • Comunicaciones comerciales: Para enviar información sobre productos, servicios y ofertas (solo con su consentimiento).
  • Mejora de servicios: Para analizar el uso del sitio web y mejorar nuestros servicios.
  • Cumplimiento legal: Para cumplir con las obligaciones legales aplicables.

3. Base Legal

El tratamiento de sus datos se basa en:

  • Ejecución de contrato: Para la gestión de pedidos y servicios contratados.
  • Consentimiento: Para el envío de comunicaciones comerciales y newsletter.
  • Interés legítimo: Para la mejora de nuestros servicios y atención al cliente.
  • Cumplimiento legal: Para cumplir con las obligaciones fiscales y contables.

4. Categorías de Datos

Tratamos las siguientes categorías de datos personales:

  • Datos de identificación (nombre, apellidos, NIF)
  • Datos de contacto (email, teléfono, dirección)
  • Datos de facturación y pago
  • Datos de navegación y uso del sitio web
  • Imágenes de referencia (para productos personalizados)

5. Destinatarios

Sus datos pueden ser comunicados a:

  • Proveedores de servicios: Empresas de transporte, procesadores de pago, servicios de hosting.
  • Autoridades públicas: Cuando sea requerido por ley.
  • No se venden ni alquilan sus datos a terceros con fines comerciales.

6. Conservación de Datos

Los datos se conservarán durante:

  • Datos de clientes: Mientras mantenga una relación comercial activa y hasta 5 años después del último contacto.
  • Datos de facturación: 6 años para cumplir con las obligaciones fiscales.
  • Datos de navegación: Hasta 2 años desde la última visita.

7. Sus Derechos

Puede ejercer los siguientes derechos:

  • Acceso: Conocer qué datos suyos tratamos.
  • Rectificación: Corregir datos inexactos o incompletos.
  • Supresión: Eliminar sus datos cuando sea legalmente posible.
  • Limitación: Restringir el tratamiento de sus datos.
  • Portabilidad: Recibir sus datos en formato estructurado.
  • Oposición: Oponerse al tratamiento de sus datos.
  • Retirada del consentimiento: Revocar el consentimiento otorgado.
